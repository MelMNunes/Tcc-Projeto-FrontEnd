# 🦶 Sistema de Gestão para Clínicas de Podologia - Front-End

Este repositório contém o código-fonte da interface web desenvolvida com foco na gestão de atendimentos em clínicas de podologia. O sistema permite que clientes, funcionários e administradores realizem ações como agendamentos, visualização de consultas, gerenciamento de fichas clínicas e muito mais.

## 🚀 Tecnologias Utilizadas

- **React** – Biblioteca principal para construção da interface
- **Next.js** – Framework para roteamento e renderização
- **TypeScript** – Linguagem com tipagem estática
- **Tailwind CSS** – Estilização rápida e responsiva
- **Axios** – Requisições HTTP com o Back-End

## 🧭 Estrutura de Perfis

- **Cliente**: realiza cadastro, agendamentos e visualiza suas consultas
- **Funcionário**: acessa atendimentos, cria anamneses e edita agendamentos
- **Administrador**: gerencia usuários e serviços
- **Recepção**: realiza agendamentos, gerencia fila e pagamentos

## 🧩 Funcionalidades

- Login e autenticação por tipo de usuário
- Cadastro de clientes e agendamentos em etapas
- Bloqueio de horários ocupados
- Histórico de consultas
- Criação e visualização de ficha de anamnese com imagem
- Pagamentos vinculados aos agendamentos
